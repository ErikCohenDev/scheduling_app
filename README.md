# Appointment Scheduling
The Appointment Scheduling App is a Java project for the C195 Class at WGU.

Main Features of the application are:
* Authentication
* I18n
* Localization (Time-Zone adjustment)
* Client Management 
    * Add
    * Modify
    * Delete
* Appointment Management
    * Notification
    * Add
    * Modify
    * Delete
* Reports
* Activity Log

Tech Stack:
- Java
